# Life2
Клеточный автомат, основанный на генетических алгоритмах.

Каждая клетка — бот, управляемый своим генетическим кодом, который может мутировать при делении.

Клетки могут:
 - фотосинтезировать
 - размножаться
 - двигаться
 - сражаться
 - отличать родственников от чужаков
 - и многое другое!
